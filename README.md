## Overview

This repository contains presentation materials from ACO482 — Design Thinking for National Security at ASU. The course applied design thinking methodology to national security problems, with a focus on a real-world collaboration with the **USAF 558th Flying Training Squadron (USAF 558)**. The project progressed through multiple iterative presentations over the semester, culminating in a final deliverable.

## Project Focus

### Design Thinking Applied to National Security
The course used the Stanford design thinking framework (Empathize → Define → Ideate → Prototype → Test) applied to a challenge relevant to the **USAF 558th Flying Training Squadron**. This type of course bridges technical cybersecurity skills with strategic and operational thinking in a defense context.

**Key themes likely covered:**
- Human-centered design in a national security context
- Problem framing for complex security challenges
- Collaborative solution development with a real stakeholder (USAF 558)
- Iterative prototyping and feedback cycles
